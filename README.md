## Getting started

### Installing

Install dependencies

```
yarn install
``` 
or
```
npm i
```

Create `.env` file with environment variables like `.env.example`.
Specify API url by setting `REACT_APP_API_URL` variable.

Starting development server

```
yarn start
``` 
or
```
npm start
```
